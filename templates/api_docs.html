GET Requests

Method/Route Description and Examples Parameters Response Type
GET /api/posts All posts in the current users' feed.
  • /api/posts
  • /api/posts?limit=5
  • /api/posts?username={{ friend.username }}
  • /api/posts?username={{ friend.username }}&limit=2
  • /api/posts?username={{ stranger.username }}&limit=2
  • username (string, optional)
    Only shows posts that were created by the requested user.
  • limit (int, optional)
    Limits the number of posts returned (defaults to 10, maximum is 50)
